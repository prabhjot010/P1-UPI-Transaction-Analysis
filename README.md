# P1: UPI Transaction Analysis

## About This Project
I took a Kaggle dataset of UPI transactions from 2024 — 
250,000 rows and 17 columns. Cleaned the data, explored 
it, answered 8 business questions, and made charts wherever 
the data needed to be visualized.

## Questions I Explored
1. Transaction status breakdown — how many succeeded, failed, or stayed pending
2. Which merchant category had the most transactions
3. Month-wise transaction amount trends
4. Weekday vs Weekend transaction patterns
5. Which state spent the most
6. Most used sender bank
7. Fraud transaction analysis — how many and in which category
8. What hour of the day sees peak transactions

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
Kaggle — UPI Transactions 2024
